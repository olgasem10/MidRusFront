Веб-сервис MidRus NLP находится в открытом доступе. Пользуясь сервисом MidRus NLP, вы соглашаетесь с тем, что полученные нами данные могут быть использованы для дальнейшего улучшения сервиса. Отзывы и пожелания приветсвуются.

Обращение к MidRus API осуществляется через веб-сервис, а также через любой инструмент веб-разработки, поддерживающий стандартные HTTP запросы и обработку документов в формате JSON.

API Reference

Для использования API необходимо отправить POST запрос на адресс *~~~~~* вместе с json-файлом формата:
{'input_text': text, 'tokenization': True, 'lemmatisation': True}, где значения для 'tokenization' и 'lemmatisation' могут быть True или False в зависимости от того, требуется ли пользователю токенизация/лемматизация или нет.

Модели

В работе API MidRusNLP используется две UDPipe-модели. Токенизация и лемматизация производится с помощью модели, обученной на коллекции данных объёмом примерно в 300 тыс. токенов. Большая часть документов выборки относится к 17 веку, однако треть данных представляет собой документы 14-16 веков, написанных на церковнославянском. Морфологическая и частеречная разметка осуществляется с помощью отдельной модели, обученной на 6,5% всего Среднерусского корпуса НКРЯ (объём около 846 тыс. токенов). Эта модель продемонстрировала отличные результаты на задачах морфологической и частеречной разметки, превысив показатели более объёмных моделей на 35%.
Метрики модели: POS-tag F1: 96.65%, Feats F1: 89.58%
