Method models

Our API uses two separate models both based on UDPipe version 1.2.1.
Tokenization and lemmatization are done with the model, trained on the collection extracted from the RNC Mid-Russian Corpus. This collection contains ~300k tokens. The Majority of douments are dated to 17th century, but nearly one third of the collection is documents from the 14-16 cc.
For morphological and pos-tagging our API uses the UDPipe model, trained on 6,5% of the whole RNC Mid-Russian Corpus (~846k tokens). This model has shown outstanding performance on these particular tasks, outperforming the larger model by almost 35%. POS-tag F1: 96.65%, Feats F1: 89.58%
