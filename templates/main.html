Middle Russian Natural Language Processing

MidRus NLP is a service for Middle Russian pos- and morphological tagging and tokenization based on UDPipe trainable pipeline. Description of the project as well as API methods is available on the About and the API Documentation web pages. In order to markup your text you can either copy it into the text area below or upload it in the CoNLL-U format.
